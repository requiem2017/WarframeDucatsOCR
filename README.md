# WarframeDucatsOCR
扫描一个文件夹下的所有截图，并生成仓库物品的白金价格以及杜卡德金币，并按照杜卡德/白金从高到低排列，扫描完成后双击/右键第一列的物品名即可复制名字方便清仓
白金价格为Warframe Market当前在线卖家前10的平均值
